# 说明

此文件夹专门处理各种纯键盘操作的功能。

此纯键盘操作的最终理想是设计一套不需要鼠标就能完成大部分操作的方案。像xmind那样

## 方案设计

### 实体选择

上下左右方向键移动当前选中的唯一实体

如果当前没有选中实体，则按下任意方向键会寻找距离屏幕中心准星直线距离最近的实体，并选中该实体

这里的实体必须是ConnectableEntity

如果当前选中了多个实体，则先取消选择所有实体，再重复上述流程。

### 生长TextNode

按下Tab键不松开，会有一个虚拟位置出现，再松开Tab键，实体会生长到该位置。

如果虚拟生长位置在一个节点上，则会生成连线。

如果虚拟生长位置在一个空白位置，则会生成一个新的节点。
