## 版本号系统

[Pride Versioning](https://pridever.org) 🏳️‍🌈 0.3.0

### 概述

当版本号为 PROUD.DEFAULT.SHAME 时，按以下规则递增：

    PROUD 版本：当你做出引以为豪的改动时递增
    DEFAULT 版本：当你发布普通/尚可的版本时递增
    SHAME 版本：当你修复难以启齿的问题时递增

支持在 PROUD.DEFAULT.SHAME 格式基础上添加预发布标签和构建元数据扩展。

### 运作原理

版本号示例：2.7.123

组成部分解析：

- 自豪版本 / 第一段
  在发布值得骄傲的版本时递增
- 默认版本 / 第二段
  普通/尚可的发布
- 羞愧版本 / 第三段
  在修复难以启齿的问题时递增

### 自豪版本递增规则

当提升自豪版本时，后续版本号将重置清零。

例如：

将 1.2.3 的自豪版本递增 1 后，会得到 2.0.0，以此充分展示新版本的自豪程度。
