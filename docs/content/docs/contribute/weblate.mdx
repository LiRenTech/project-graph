## weblate是什么？

Weblate是一个基于Web的自由软件持续本地化系统，专为开源项目国际化（i18n）设计。它通过Git版本控制无缝集成翻译流程，支持.po、.json、.yaml等30+文件格式，让全球贡献者能在统一界面协作翻译，同时保持译文与代码库的实时同步

## 选择Weblate对贡献翻译者有什么好处？

无需掌握Git命令，网页端可视化界面可直接编辑翻译条目，支持实时预览上下文

集成术语库强制校验、拼写检查等质量控制工具

贡献者档案页自动统计翻译字数与活跃度，优秀译者可晋升为校对员参与质量把关。所有翻译行为均通过Git提交记录永久留存，成为开发者履历中的可验证开源贡献

## 此项目为何使用Weblate？

由于随时可能会增加功能，进而出现新的设置项、UI界面等。如果每次做新功能，开发者就需要手动编辑其他语言的翻译文件，精力不足。

即使是精力充足的情况下，也难免会出现漏翻译的情况，Weblate可以提供遗漏检查功能
