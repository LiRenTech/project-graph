### 安装 Rust 和 Node.js

- 安装 Rust：https://www.rust-lang.org/tools/install
- 安装 Node.js：https://nodejs.org/en/download/
- 安装 pnpm：`npm install -g pnpm`

### 安装项目依赖

```sh
pnpm install
```

### 启动应用

```sh
# 桌面版
pnpm dev:app
# 网页版
pnpm dev:web
# 文档
pnpm dev:docs
# Android 版
pnpm dev:android
```

### 提交更改

- 安装 Gitmoji: `pnpm install -g gitmoji-cli`
- 提交更改: `gitmoji commit`
